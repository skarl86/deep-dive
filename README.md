# 🚀 Deep Dive - 프론트엔드 학습 저장소

> **Next.js + React**를 활용한 실전 프로젝트 기반 학습 저장소

이 저장소는 최신 프론트엔드 기술 스택을 체계적으로 학습하고 실무에 적용하는 것을 목표로 합니다.

---

## 📂 프로젝트 목록

### [🎬 Movie Database](/movie)

**Next.js 16 + React 19** 기반 영화 데이터베이스 애플리케이션

**핵심 학습 내용:**

- Next.js 16 App Router 및 Server/Client Component 전략
- Zod를 활용한 타입 안전성 및 데이터 검증
- Playwright를 활용한 E2E 테스트 시스템
- 웹 접근성(A11y) WCAG 2.1 AA 준수
- 성능 최적화 및 캐싱 전략

**기술 스택:** Next.js 16, React 19, TypeScript, Tailwind CSS 4, shadcn/ui, Zod, Playwright

---

## 🎯 학습 목표

- ✨ **최신 프레임워크 마스터**: Next.js, React 최신 버전 완벽 이해
- 🔐 **타입 안전성**: TypeScript + Zod를 활용한 엔드투엔드 타입 안전성
- 🧪 **테스트 주도 개발**: E2E, 접근성, 시각적 회귀 테스트 체계 구축
- ♿ **웹 접근성**: WCAG 기준 준수 및 포용적 디자인
- 🚀 **성능 최적화**: 실무 수준의 성능 최적화 기법 습득
- 🤝 **AI 페어 프로그래밍**: Cursor AI를 활용한 효율적 개발 방법론
